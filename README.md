# church-stage-timer

> Church / Stage timer Electron App

## Install

Install with [npm](https://www.npmjs.com/)

```sh
$ npm i church-stage-timer --save-dev
```

## Usage

```js
var church-stage-timer = require('church-stage-timer');
```

## Running tests

Install dev dependencies:

```sh
$ npm i -d && npm test
```
